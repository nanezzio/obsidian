### Понятие 

F-строки - это "форматированные строки литералы", которые позволяют вставлять в строку значения переменных и произвольные выражения внутри фигурных скобок

### Синтаксис

Обычная строка в которой стоит префикс f;
В них пишется обычное выражение в `{}` - они будут вычислены и подставлены в результирующую строку

### Примеры:

1#
```python
name = "Степан"
age = 18
s = f"Меня зовут {name}, мне {age} лет"
# Меня зовут Степан, мне 18 лет
```

2#
```python
form math import pi 
s = f"pi округленное до двух знаков: {pi:.2}"
# pi округленное до двух знаков: 3.14
```

f-строка разворачивается 1 раз - в момент ее создания, после этого результат обычная строка, которая уже не меняется при изменении переменных. Внутри `{}` можно использовать почти любое корректное выражение: сложение, вызов функции, методы объектов, условные выражения 

```python
x = 2
y = 21
f"{x * y}"   # '42'
f"{name.upper()}"   # 'СТЕПАН'
f"{round(13/3)}"   # '4'
f"{'четное' if x % 2 == 0 else 'нечетное'}"   # 'четное'
```

**НЕЛЬЗЯ** напрямую использовать в f-строке обратные фигурные скобки без экранирования, потому что `{` и `}` - служебные символы синтаксиса
Пример экранирования:

```python
f"{{пример}} = {x}"   # '{пример} = 2'
```
то есть их нужно удвоить, чтобы вывести `{}`


### Форматирование: числа, выравнивание, ширина

Общий вид `{expr:формат}`

После выражения в `{}` можно указать двоеточие `:` и формат-строку - это тот же мини-язык, который используется в методе [[str.format]]

Основные элементы формата:

- Выравнивание и ширина: `>` - вправо, `>` - влево, `^` - по центру, число - минимальная ширина поля
- Заполнитель: символ перед выравниванием, например `0>8` - заполнить нулями слева до ширины 8
- Для чисел:
	- `d` - целое число, `f` - число с плавающей точкой, `e` - экспоненциальная форма
	- `.2f` - 2 знака после запятой, `.3e` - 3 знака в экспоненциальной записи
	- `,` - разделитель тысяч

Примеры:

```python
n = 1234.56789
f"{n:.2f}"     # '1234.57'
f"{n:,.2f}"    # '1,234.57'
f"{n:10.2f}"   # '   1234.57' (ширина 10, по умолчанию вправо)
f"{n:^10.2f}"  # ' 1234.57 ' (по центру)
```


### Форматирование дат и других объектов

Любой объект, для которого определен метод `__format__`, может использоваться с формат-спецификацией. Для `datetime` можно использовать стандартные коды [[strftime]]

Пример:

```python
from datetime import datetime
now = datetime.now()
f"{now:%Y-%m-%d %H:%M}"   # '2026-03-03 21:39'
```


### Отладочный выводка (only Python 3.8)

Синтаксис `{x = }`

Автоматически подставляет текст выражения, знак = и его значение

Пример:

```python
x = 10
y = 25
f"{x = }, {y = }" # 'x = 10, y = 25'
```

Можно комбинировать с форматированием


### Конверсия `!r`, `!s`, `!a`

Внутри `{}` можно использовать конверсию перед двоеточием: `!r`, `!s`, `!a`
- `!r` - применяет `repr(obj)` (обычно "техническое" представление, полезно для отладки)
- `!s` - `str(obj)` (человеко-читаемый вид, используется по умолчанию)
- `!a` - ascii(obj) (экранирует не-ASCII)
