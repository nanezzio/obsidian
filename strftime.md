### Введение 

`strftime` (`string format time`) - это способ превратить дату\время в строку по заданному формату. В Python он есть как функция в модуле `time`, и как метод у объектов `datetime`

1.  Общая идея
Есть объект времени (например, `datetime.now()`), а нужно получить строку вроде `"2026-03-03 22:15"` или `"03.03.2026"`. Для этого нужно вызвать `strftime` и передать строку-шаблон с особыми кодами:

```python
from datrtimr import datetime

now = datetime.now()
s = now.strftime("%Y-%m-%d %H:%M:%S")
print(s) # например: '2026-03-03 22:15:07'
```
