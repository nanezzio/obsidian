### Введение 

`strftime` (`string format time`) - это способ превратить дату\время в строку по заданному формату. В Python он есть как функция в модуле `time`, и как метод у объектов `datetime`

1.  Общая идея
Есть объект времени (например, `datetime.now()`), а нужно получить строку вроде `"2026-03-03 22:15"` или `"03.03.2026"`. Для этого нужно вызвать `strftime` и передать строку-шаблон с особыми кодами:

```python
from datrtimr import datetime

now = datetime.now()
s = now.strftime("%Y-%m-%d %H:%M:%S")
print(s) # например: '2026-03-03 22:15:07'
```

2. Синтаксис
Есть два основных варианта:

```python
import time
time.strftime(format, t)   # функция модуля time

from datetime import datetime
dt.strftime(format)   # метод объекта datetime
```

- `format` - строка с кодами формата (`"%Y-%m-%d"`)
- `t` - структура времени (`time.struct_time`), обычно берут из `time.localtime()`; если не передать, берется текущее время

Пример с модулем `time`:

```python
import time

s1 = time.strftime("%Y-%m-%d", time.localtime()) 
# '2020-04-24' в примере из документации 

s2 = time.strftime("%d.%m.%Y г. %H:%M", time.localtime()) 
# '24.04.2020 г. 19:26'
```

Пример с `datetime`:

```python
from datetime import datetime 

dt = datetime(2025, 12, 31, 12, 59, 59) 
s = dt.strftime("%Y-%m-%d") 
# '2025-12-31'
```


3. Основные коды формата

- `%Y` - год полностью: `2026`
- `%y` - год двумя цифрами: `26`
- `%m` - месяц с ведущим нулем: `01` - `12`
- `%d` - день месяца с ведущим нулем: `01` - `31`
- `%H` - часы в 24-часовом: `01` - `23`
- `%I` - часы в 12-часовом: `01` - `12`
- `%M` - минуты: `00` - `59`
- `%S` - секунды: `00` - `59`
- `%A` - полное название дня недели (в локали, например `Monday`)
- `%B` - полное название месяца (`November`)
- `%j` - день года: `001` - `366`
- `%c` - дата и время в "локальном" формате
- `%x` - дата в локальном формате (обычно `MM/DD/YY`)
- `%X` - время в локальном формате
- `%%` - символ `%` в результирующей строке

Пояснения:
- «Ведущий ноль» - это ноль в начале числовой строки, который добавляется специально, чтобы длина числа была фиксированной (обычно 2 символа)
- «Локальный формат»  - это формат даты и времени, зависящий от настроек локали (языка и региональных настроек системы или программы)
- Краткое название библиотеки `from ... import ... as <название>`
Примеры:

```python
from datetime import datetime as dt

now = dt.now()

day = now.strftime("%d") # '10'
month = now.strftime("%B") # 'November'
year = now.strftime("%Y") # '2022'
time_str = now.strftime("%H:%M") # '15:40'

print("Today is")
```
